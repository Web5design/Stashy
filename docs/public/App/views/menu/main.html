Menu

Hidden Menu for you applications

  Download   View project on Github

Description

This component allow to create hidden navigation menus.

The component will be hidden on the initial load of the application and will only show a small tab.Clicking on the tab will show the menu.

You can enable drag to show or hide the menu.

Javascript

Constructor

Stashy.Menu(selector, userOptions)
Parameter Type Description
selector string CSS selector
userOptions object The initial options for each instance of the menu. Take a look to the userOptions table and explore the available options.

The userOptions object

Option Type Default Description
enableTouch bool false Allow to open/close menu using drag events. It depends on including jquery.hammer among your scripts.
closeOnClick bool false Close menu when click on any option.
closeOnClickOutside bool false Close when clicking outside the menu container.

Methods

Name Description
on() Always call this method after creating the instance for the initial layout and bindings
open() Open the menu
close() Close the menu

Sample

var menu = Stashy.Menu(".st-menu", { enableTouch : true, closeOnClick : true, closeOnClickOutside : true}).on();

$("#open-btn").on("click", function() {
    menu.open();
    return false;
});

$("#close-btn").on("close", function() {
    menu.close();
    return false;
});
